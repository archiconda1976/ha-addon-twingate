# Home Assistant Twingate Connector Add-on

Run a Twingate Connector directly inside Home Assistant.

## Install

1. Settings → Add-ons → Repositories
2. Add:
   https://github.com/archiconda1976/ha-addon-twingate
3. Install "Twingate Connector"

## Integration

Copy custom_components/twingate_connector into your Home Assistant config folder.
Restart Home Assistant.
